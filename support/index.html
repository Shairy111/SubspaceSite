Support

Help for Subspace Subscription Tracker.

Subspace is a local-first macOS app for tracking subscriptions, renewals, trials, and Apple-native reminders. This page covers practical help for setup, Apple integrations, purchases, and App Store review support.

Getting help

Report an issue or ask a question.

The fastest support path is to open a GitHub issue with your macOS version, Subspace app version, and a short description of what happened.

Please avoid sharing private billing information, full receipts, or sensitive account details in public support requests.

Before reporting

Helpful details to include

  • The Subspace version shown in the app or App Store.
  • Your macOS version and whether you are using iCloud, Calendar, or Reminders sync.
  • What you expected to happen and what actually happened.
  • Any non-sensitive error text shown by the app.

Common setup help

Apple integrations

iCloud sync
iCloud sync requires a signed-in Apple Account with iCloud available on the Mac. If sync is unavailable, check macOS iCloud settings, make sure Subspace has the required iCloud capability in the App Store build, and try the app's repair or resync action when available.
Apple Calendar sync
Calendar sync is optional. If permission is denied or blocked, open macOS System Settings and allow Calendar access for Subspace, then return to Subspace settings and retry sync.
Apple Reminders sync
Reminders sync is optional. If permission is denied or blocked, open macOS System Settings and allow Reminders access for Subspace, then retry from Subspace settings.
Local notifications
Renewal and trial alerts use local macOS notifications. If notifications do not appear, check macOS notification settings for Subspace and confirm notification reminders are enabled in the app.

Billing

Subspace Pro purchases and refunds

Subspace Pro subscriptions are purchased and managed through Apple's App Store and StoreKit. If Pro access does not appear after purchase, use Restore Purchases in Subspace and make sure you are signed into the Apple Account that made the purchase.

Apple handles payment processing, subscription management, cancellations, and refund requests. To manage or cancel a subscription, use Apple's subscription management surface from macOS, the App Store, or Subspace when the manage-subscription action is available.

Data model

Subspace is local-first.

Your subscription library is owned by Subspace and stored locally first. Optional iCloud, Calendar, Reminders, and notification features are projections or sync helpers that you control from app settings.

App Review

Review notes

Subspace does not require a developer-run account to use the core local library. Pro features use Apple StoreKit subscriptions, and Apple-native integrations ask for system permissions only when the user enables those features.
